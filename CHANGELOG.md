# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/)
y este proyecto se adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-12
### Añadido
- Lanzamiento principal de **Ventura Boutique**.
- Estética **Neo-Cyber** interactiva con tipografía Outfit y Questrial.
- Sistema de autenticación simulada (Mail/Google) con persistencia en localStorage.
- Catálogo de productos atemporal con imágenes generadas por IA.
- Gestión de Carrito de Compras y Favoritos.
- Panel de Control del Usuario con historial de pedidos y ajustes.

## [0.2.0] - 2026-04-12
### Añadido
- Archivo de prueba `TEST.md` para validación de flujo de trabajo.

## [0.1.0] - 2026-04-12
### Añadido
- Inicialización del repositorio Git.
- Configuración de archivo `.gitignore` con patrones para Node.js, IDEs y agentes.
- Estructura base de skills (`commiter`, `skill-creator`).
